User Guide


Loading and labeling scenes
  • Open the menu and click 'Auto get config' to load a scene with objects.
  • Move the objects (selected object will be red, others will be gray) so that they match the ones in the scene
  • Open the menu and click 'Finalize match' to submit the results
Controlling the Scene:
  • To rotate the scene, right click and drag.
  • To translate the scene, left click and drag.
Controlling the Object:
  • Click on an object to select it. The selected object will be red
  • To rotate the object, hold down 'x', right click and drag.
  • To translate the object, hold down 'x', left click and drag.
Other controls:
  • To run the auto-snapping feature, press 'i'
  • To run the sanity check feature, press 't'. The currently selected object will be drawn into a photograph of the scene. Press 't' again to close the image.
